    Paragraphs

    • Paragraph banane ke liye p tag use hota hai.

    • Ek ya zyada sentences ko ek relevant group mein format karta hai.
    • Example:

      Yeh ek paragraph hai.

    Headings

    • Headings banane ke liye h1 se h6 tak tags use hote hain.

    • Heading ka size h1 se h6 tak chhota hota hai.
    • Example:

      Yeh ek heading hai

    Lists

    • Sab lists mein { li } (list item) tag use hota hai.
    • Ordered List: {ol} tag ke andar {li} items hote hain, jo numbered sequence mein dikhte hain.
    • Unordered List: {ul} tag ke andar {li} items hote hain, jo bullets ke saath dikhte hain.

    Ordered List

    1. type="a": Lowercase letters (a, b, c).
    2. type="A": Uppercase letters (A, B, C).
    3. type="i": Lowercase Roman numerals (i, ii, iii).
    4. type="I": Uppercase Roman numerals (I, II, III).

    Unordered List

    • type="disc": Default bullet (solid circle).
    • type="circle": Hollow circle.
    • type="square": Solid square.
